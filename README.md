# Machine Learning Exercises  
**Implemented by Luluh Almogbil**

This repository contains a set of **machine learning notebooks and exercises** built for learning and experimentation using **scikit-learn**.

The focus is on understanding data, building predictive pipelines, and evaluating models using different matrices to compare and learn more about my models

---

## This Repository Covers

- Dataset exploration and feature understanding  
- Numerical vs categorical features  
- ML pipelines with scikit-learn  
- Overfitting, underfitting, and generalization  
- Cross validation and model evaluation  
- Hyperparameter tuning and comparison using grid search and more
